Auth Authentication

Next Auth v5 - Advanced Guide (2024) offers extensive authentication features using Next-auth v5 and Next.js 14. Key components include login, registration, and password recovery. It supports social logins via Google & GitHub, email verification, and two-factor authentication. User roles, admin-only access, and customizable settings enhance security. This project streamlines authentication implementation in Next.js applications.

App Daily Goal

This app features a homepage displaying a list of habits, each showcasing the past 7 days' progress with checkboxes for completion, "x" for incomplete, or gray circles for unmarked days. It includes functionality to delete habits, redirect to habit details upon selection, and add new habits through a separate page. The habit details page exhibits the habit name with a back button. Additionally, a calendar view allows navigation between months, enabling users to mark habits' completion status per day. Deployment is achieved using Vercel.

Additionally, it includes functionality for both clients to schedule appointments and administrators to manage bookings.
